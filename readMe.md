#使用方法

-	基于apicloud开发，请同步至你的apicloud工程后编译
-	ajax请求，返回json数据
>	成功返回 
	`{status:1,error:0,data:[...]}
>	失败返回
	`{status:0,error:msg}
-	数据端使用php返回json
